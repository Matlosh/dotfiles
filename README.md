# Mat's dotfiles

This repo contains a collection of my most important dotfiles and simple daily use scripts.

## Installation

To install, simply clone the repo and place files in their corresponding locations (you should assume that all directories and files should be placed relative to user's home directory).

```bash
git clone https://github.com/Matlosh/dotfiles.git
```

## Structure

- .bashrc - exports, aliases and settings to make basic tools work in the terminal
- scripts/ - collection of simple daily use scripts
- .config/ - configuration files for various applications
  - nvim/ - Neovim configuration
  - hypr/ - Hyprland (main graphical envrionment) configuration
  - kitty/ - Kitty terminal emulator configuration
  - waybar/ - Waybar (top bar) configuration
